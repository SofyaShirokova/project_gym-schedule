# Gym Schedule Management System

Веб-приложение для управления расписанием тренировок спортивного зала на основе Flask. Позволяет спортсменам записываться на тренировки, тренерам создавать занятия и администраторам управлять расписанием и доступностью мест.

## Установка

Для запуска проекта необходимо:

- Python 3.7 или выше
- pip (менеджер пакетов Python)

Установка зависимостей:

    pip install flask pandas

Инициализация базы данных:

    python get_database.py

Запуск приложения:

    python app.py

После запуска приложение будет доступно по адресу: http://127.0.0.1:5000

## Пример работы приложения

### Главная страница (`/`)
- Выбор существующего спортсмена или регистрация нового
- Просмотр записей на тренировки выбранного спортсмена
- Возможность записи на новые тренировки
- Подтверждение посещения или отмена записи

### Создание тренировки (`/info_t`)
Создание нового занятия с указанием:
- Описания тренировки
- Типа тренировки (из списка)
- Дня недели
- Продолжительности
- Количества доступных мест
- Назначенного тренера

### Поиск тренировок (`/search`)
Фильтрация тренировок по:
- Тренеру
- Типу тренировки
- Дню недели

### Регистрация спортсмена (`/new_sportsmen`)
Добавление нового спортсмена в систему

## Структура проекта

Основные файлы:
- `app.py` - основной файл приложения Flask
- `SportZal.sqlite` - база данных SQLite
- `get_database.py` - скрипт инициализации БД
- `models/` - модели данных и бизнес-логика
- `controllers/` - обработчики запросов
- `templates/` - HTML-шаблоны
